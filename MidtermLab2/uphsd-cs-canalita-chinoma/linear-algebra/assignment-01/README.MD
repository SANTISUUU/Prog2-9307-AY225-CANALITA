## Project Description
This program calculates the determinant of a **3×3 matrix** using the **cofactor expansion method along the first row**.

The matrix values are **hardcoded in the program**, and the program prints the **step-by-step computation** including:

- The original matrix
- Each **2×2 minor**
- The **cofactor calculations**
- The **final determinant value**

This allows the user to clearly follow how the determinant is calculated.
